# Deployment States

### Deployment States

Deployments go through a series of states. They are enumerated here & are useful if querying the status of deployment via the API.

| ID | Name |
| :--- | :--- |
| 1 |  `Building`  |
| 2 | `Provisioning` |
| 3 | `Starting` |
| 4 | `Running` |
| 5 | `Stopping` |
| 6 | `Stopped` |
| 7 | `Error` |
