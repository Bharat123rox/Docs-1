# Deployment States

### Deployment States

Deployments go through a series of states. They are enumerated here and are useful if you you're querying the status of a Deployment via the API.

| ID | Name |
| :--- | :--- |
| 1 |  `Building`  |
| 2 | `Provisioning` |
| 3 | `Starting` |
| 4 | `Running` |
| 5 | `Stopping` |
| 6 | `Stopped` |
| 7 | `Error` |
